# can-rent-lah 🏠🇸🇬

> Can rent lah, don't worry.

AI-powered Singapore property search CLI. Built on [opencli](https://github.com/jackwener/opencli). For international students and anyone hunting for a roof in the Little Red Dot.

```
$ can-rent search "clementi mrt" --max 2500 --bedrooms 2
```

## What?

opencli makes any website a CLI. `can-rent-lah` adds PropertyGuru support so your AI agent can search Singapore properties for you — MRT proximity, HDB vs Condo, budget filters, the works.

Chope your dream home from the terminal.

## Setup

1. Install [opencli](https://github.com/jackwener/opencli)
2. Chrome running & logged into [propertyguru.com.sg](https://www.propertyguru.com.sg)
3. Register: `opencli external install <path-to-this-repo>/cli/propertyguru`
4. `/can-rent-lah-setup` in your agent

## Skills

| Skill | What |
|-------|------|
| `/can-rent-lah-setup` | Set up opencli + register PropertyGuru CLI |
| `/can-rent-lah-search` | Search Singapore properties with filters |

## Why "can-rent-lah"?

"Can" is the most Singapore word. "Lah" is the most Singapore particle. Say it together and you've already rented the place.
